Una disculpa tenia que atender unos asuntos.

Mire por lo general se cobra por renta o por venta al ser un desarrollo nuevo, en desarrollos empezados (features nuevos, cambios de estilos, etc) se cobra por hora o por paquete lo cual no es su caso, usted desea migrar un sistema que ya tiene a uno web (tecnologias mas robustas), lo cual es un desarrollo desde cero o nuevo.

Los desarrollos web ocupan de pagar una serie de servicios para que estos funcionen y esten en condiciones productivas (independiemente del pago al programador), estos servicios cubren desde el nombre de dominio que usted desea tener para su sitema (www.ejemplo.com), el hospedaje o tambien llamado hosting (tanto del frontend como del backend) el cual permite ser accesible su sistema a los usuarios, la base de datos (muchas veces incluida en el hosting siempre que no rebase cierto limite), entre otros gastos que dependen de los proveedores de estos servicios.

Por darle un ejemplo con proveedores que yo suelo usar:

Heroku (hosting Backend) : $140 MXN mensuales para proyectos pequeños a $1000 MXN mensuales para un proyecto profesional mediano.
Heroku (base de datos incluida): Gratis o $1000 rebasando niveles de almacenamiento gratuito.
Firebase (hosting Frontend): Gratis (esto puede cambiar pero ah sido gratis por varios años ya).
Dominio (nombre): Precios promedio en México (anual):
                  .com: $199 - $350 MXN
                  .mx: $120 - $600 MXN
                  .com.mx: $100 - $350 MXN
                  .net / .org: $185 - $450 MXN 

Ahora bien dejeme le explico las dos modalidades de cobro:

Venta:
Un proyecto por venta yo realizo el desarrollo, una vez este cumpla con sus necesidades se pondra a disposición de usted, se entregaria el codigo, se genera un repositorio (una bodega en la nube que contiene el codigo tambien), si gusta yo hago un despliegue inicial dejando todo productivo con un costo pequeño y usted se encarga de todo los tramites posteriores, pagos a dichos servicios, monitoreo, mantenimiento, etc. Cabe aclarar que todo aquello que ocupo cuenta y contraseña se le entrega a usted ya se encarga de gestionarlo. Aqui usted tiene el control total y propiedad intelectual del sistema

Por ejemplo:
  Proyecto básico/Freelance: Entre $15,000 y $25,000 MXN.
  Proyecto a medida (PyME): Entre $20,000 y $50,000 MXN.

Renta:
Un proyecto por renta yo realizo el desarrollo, una vez este cumpla con sus necesidades se pondra en estado productivo, yo me encargare de los pagos, los despligues, monitoreos, si el servicio llegase a fallar veria la forma de levantarlo, etc. Aqui yo cobraria uan renta fija mia y los precios variables de los servicios de terceros.

Por ejemplo:
Renta fija mensual de $2000 + costos variables

Todos estos precios son provisionales y para fines de entendimiento, tanto la venta como la renta se acoplarian a su presupuesto y sus necesidades del proyecto.

Ya si estos precios le parecen bien podemos empezar a ver ahora si que ocupa su sistema por completo, hacer un diagrama de relaciones de tablas para entender bien que es lo que necesita, ir desglozando cada necesidad, ya poder darle una recomendación de tecnoligias (esto afecta el costo puede verse un ahorro o un aumento de precio por parte de los servicios dependiendo que se elija) y cotización real en base a todo lo previamente mencionado, posteriormente si todo va bien yo empezaria con el desarrollo.
